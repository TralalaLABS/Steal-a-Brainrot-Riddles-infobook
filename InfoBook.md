# Overview

Steal a Brainrot is a Roblox game focused on collecting, stealing,
and upgrading Brainrots.

Brainrots can have:

- Mutations
- Traits
- Different rarities
- Different values
- Limited availability

Some content is permanently unavailable after events.

---

# Value Formula

Brainrot income is calculated using:

    Final Value =
    (Brainrot Value × Mutation Multiplier)
    + Trait Multiplier 1
    + Trait Multiplier 2
    + Other Trait Multipliers

Multiple traits can stack.

Some traits use special calculations.

Example:

Ventoliero Pavonero:

Base income:
$65M/s

Default:
1x multiplier

Gold:
1.25x multiplier

Cyber:
11x multiplier


With one 5x trait:

Default:
6x

Gold:
6.25x

Cyber:
16x


With two 5x traits:

Default:
11x

Gold:
11.25x

Cyber:
21x
