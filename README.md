# SAB Info Book

A community-made **Steal a Brainrot (SAB)** information book.

This repository contains a Markdown reference file with:

- OG Brainrots
- Mutations
- Mutation multipliers
- Traits
- Trait multipliers
- Events
- Secret Brainrots
- Trivia
- SAB history notes

---

# How To View

## View Online (Recommended)

1. Open this repository on GitHub.
2. Click:

`SAB Info Book.md`

3. GitHub will automatically display it as a formatted book.

---

# Download The File

## Download From GitHub

1. Open:

`SAB Info Book.md`

2. Click the **Raw** button.

3. Save the page as:

`SAB Info Book.md`

# Download Repository

You can download the entire repository:

1. Click the green **Code** button.
2. Click:

`Download ZIP`

3. Extract the ZIP file.
4. Open:

`SAB Info Book.md`
